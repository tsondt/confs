#My dotfiles

##Requirements
- `bash`
- `git`
- `tmux`
- `vim` (or `neovim`)

- Install `vim-plug` (https://github.com/junegunn/vim-plug#installation)
  Vim
  ```
  curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
      https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
  ```
  Neovim
  ```
  curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
      https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
  ```

- Then run:
  
  ```
  ./install.sh
  ```
